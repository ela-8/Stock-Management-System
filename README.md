# Stock Management System (A-Level NEA)
A desktop-based stock management system developed using Visual Basic to help small cafés efficiently manage inventory.

## Features
- Add and update stock items
- Process customer orders (automatic stock reduction)
- Low stock alert system
- Daily and weekly usage tracking
- Search and delete stock items
- User authentication with password hashing (SHA256)
- Data persistence using file storage (CSV)

## Technical Concepts Used
- Object-Oriented Programming (Item class)
- Lists (List(Of Item))
- File handling (Read/Write CSV)
- Input validation (Integer.TryParse, range checks)
- Searching algorithms (linear search)
- Password hashing (SHA256)
- Multi-form navigation system

## Technologies
- Visual Basic (.NET)
- Windows Forms

## Project Context
This project was developed as part of my A-Level Computer Science NEA.  
It focuses on solving real-world inventory management problems for small businesses.
